# 角色状态

角色状态是对于行动顺序表而言的行动状态，优先级最高，决定了角色是否可以行动。

## 状态类型

### 控制态

影响角色行动和决策的状态，高级别覆盖低级别：

| 状态 | 效果 | 允许操作 |
|---|---|---|
| **完全行动不能** | 回合被跳过 | 无 |
| **行动受限** | 决策受限 | 仅特殊物品 |
| **战斗不能** | 无法战斗 | 仅物品 |
| **技能受限** | 无法使用技能 | 普攻 + 物品 |
| **攻击受限** | 无法普攻 | 技能 + 物品 |

> 当同时存在技能受限和攻击受限时，被**战斗不能**覆盖。

### 吟唱态

归行动顺序表管理：

| 状态 | 说明 |
|---|---|
| **吟唱中** | 正在吟唱魔法，行动阶段自动结算，无法手动决策 |
| **预释放爆发技** | 已预释放爆发技，需选择目标，无法取消或执行其他行动 |

吟唱中被施加控制态（攻击受限除外）→ 打断施法，取消吟唱态。

预释放爆发技是**最高级别**状态，无法被任何状态覆盖（除非死亡）。

### 常态

允许角色自主决策，这是默认状态。

## 状态覆盖优先级

```
预释放爆发技
    ↓
完全行动不能
    ↓
行动受限
    ↓
战斗不能
    ↓
技能受限 / 攻击受限
    ↓
吟唱中（可被打断）
    ↓
常态
```
