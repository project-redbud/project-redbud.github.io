# 核心属性

角色拥有 **力量、敏捷、智力** 三大属性。角色选择其中一个作为**核心属性**，是数值计算的重要参考。

属性由 **基础属性** 和 **额外属性** 相加而成。

---

## 攻击力提升

每提升 1 点基础核心属性 → 提升 1 点基础攻击力  
每提升 1 点额外核心属性 → 提升 1 点额外攻击力

---

## 力量

| 每 1 点力量 | 加成 |
|---|---|
| 最大生命值 | +13 |
| 生命回复 | +0.15 |
| 物理护甲 | +0.75 |
| 暴击伤害 | +0.575% |

## 敏捷

| 每 1 点敏捷 | 加成 |
|---|---|
| 行动速度 | +0.65 |
| 闪避率 | +0.175% |
| 暴击率 | +0.25% |

## 智力

| 每 1 点智力 | 加成 |
|---|---|
| 最大魔法值 | +8 |
| 魔法消耗减少 | 0.125% |
| 爆发能量消耗减少 | 0.075% |
| 加速系数 | +0.125% |
| 冷却缩减 | +0.25% |
| 魔法回复 | +0.1 |
