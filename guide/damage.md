# 伤害计算

## 六乘区模型

FunGame 的伤害计算分为 **6 个乘区**，按以下顺序计算：

```
① 基础伤害倍率
   ↓
② 伤害加成区间 1
   ↓
③ 减伤
   ↓
④ 暴击伤害区间
   ↓
⑤ 伤害加成区间 2
   ↓
⑥ 魔法效能（仅魔法技能）
```

### ① 基础伤害倍率

```
基础伤害 = 基础伤害基底 × 基础伤害倍率
```

例如普通攻击基于 100% 攻击力，则取攻击力的数值 × 100%。

### ② 伤害加成区间 1

- **固定值加成**：直接加法 `基础伤害 + 100`
- **百分比加成**：`基础伤害 × (1 + 所有百分比之和)`

::: tip 示例
两个技能分别提供 30% 和 70% 加成：
`基础伤害 × (1 + (30% + 70%)) = 基础伤害 × 2`
:::

### ③ 减伤

```
基础伤害 = 基础伤害 × (1 - 减伤%)
```

若目标减伤为 20%：`基础伤害 = 基础伤害 × 0.8`

### ④ 暴击伤害区间

```
基础伤害 = 基础伤害 × 暴击伤害倍率
```

### ⑤ 伤害加成区间 2

计算方式同区间 1。

### ⑥ 魔法效能

仅魔法技能应用此乘区，详见技能系统 → 魔法。

## 物理伤害计算

以普通攻击为例的完整流程：

### 1. 预计伤害

```
(基础攻击力 + 附加攻击力) × 伤害系数
```

### 2. 物理穿透

削减目标的物理护甲。角色穿透 30%，目标护甲 60：

```
(1 - 0.3) × 60 = 42 （实际护甲）
```

### 3. 物理伤害减免

```
物理伤害减免 = 护甲 / (护甲 + 240) × 100%
```

接上例：`42 / (42 + 240) = 14.89%`

### 4. 最终物理伤害

```
预计伤害 × (1 - 物理伤害减免)
```

## 魔法伤害计算

与物理伤害类似：

| 物理 | 魔法 |
|---|---|
| 物理伤害减免 | 魔法抗性 |
| 物理穿透 | 魔法穿透 |
| 护甲 → 减免转换 | 穿透直接作用于抗性 |

魔法伤害具有属性，不同属性应用不同魔法抗性，但魔法穿透为共用属性。
