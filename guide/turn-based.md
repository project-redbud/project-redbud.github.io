# 回合制系统

FunGame 采用**完全回合制**，角色按顺序行动，只有先前角色行动结束后，才轮到下一个角色。

## Round 与 Turn

回合分为 `Round` 和 `Turn` 两种表示：

| 概念 | 含义 |
|---|---|
| **Round** | 游戏回合。每个角色的一次行动，都是一个 Round |
| **Turn** | 角色轮次。角色每次轮到行动，即为该角色的一个 Turn |

> ⚠️ **关键区分**：技能特效计算的回合数，以角色的 Turn 为准。由于顺序表存在插队机制，每个角色的 Turn 数是不同的。

## 回合外

任意角色的回合，对于其他角色而言都是**回合外**。

本质上行动顺序是一个环形：

```
回合外 → 回合开始 → 行动阶段 → 回合结束 → 回合外
```

## 回合内

回合内分为以下几个阶段：

### 准备阶段

准备阶段会检查角色的状态：

- 角色处于 **控制态** → 行动决策受影响，甚至可能跳过回合
- 角色处于 **吟唱态** → 结算吟唱的技能，决策受限

### 行动阶段

处于 **常态** 的角色可以进行决策：

| 行动 | 说明 |
|---|---|
| 普通攻击 | 基于攻击力的物理伤害 |
| 释放战技 | 消耗爆发能量 |
| 吟唱魔法 | 使角色进入吟唱态（结束回合） |
| 使用物品 | 消耗品/装备 |
| 放弃行动 | 本轮不操作 |

### 结束阶段

此阶段进行各种结算：硬直时间计算、时间流逝、状态持续时间更新等。

## 时间

回合制同样存在**时间 (Time)**。

### 硬直时间 (Hardness Time)

也称行动延迟 (Action Delay)：**角色无法行动的时间，或下次行动所需的等待时间**。

- 没有角色行动时 → 时间持续流逝，直到有角色硬直时间衰减至零
- 角色行动回合内 → 时间暂停
- 每当时间流逝 → 减少硬直时间、冷却时间、特效持续时间

> 时间没有确切单位，可用"秒"或"单位时间"泛指。每次计算保留两位小数。
