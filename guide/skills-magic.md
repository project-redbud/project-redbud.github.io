# 魔法

魔法消耗魔法值。通常来源于角色装备的魔法卡包，部分角色初始自带魔法。

## 魔法流程

```
行动回合 → 选择魔法 → 选择目标 → 开始吟唱 → 结束回合
   ↓
回合外 → 吟唱结束 → 发动效果
   ↓
回合外 → 硬直时间衰减 → 下次行动回合
```

## 等级

| 来源 | 初始 | 最高 | 升级方式 |
|---|---|---|---|
| 角色自带 | 1 级 | 8 级 | 等级解锁 (8/16/24/32/40/48/56) |
| 魔法卡包 | 卡包决定 | — | 魔法卡升级（无等级要求） |

## 魔法瓶颈

部分魔法对角色的智力有最低要求，称为**魔法瓶颈**。

## 魔法效能

```
魔法效能 = 1.0 + (角色智力 - 魔法瓶颈) ÷ 魔法瓶颈
```

| 智力 vs 瓶颈 | 魔法效能 | 实际效果 |
|---|---|---|
| 智力 = 瓶颈 | 100% | 标准效果 |
| 智力 < 瓶颈 | < 100%（最低 1%） | 效果削弱 |
| 智力 > 瓶颈 | > 100%（最高 200%） | 效果增强 |
