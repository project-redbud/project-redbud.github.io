# 战技 & 被动

## 主动战技

简称**战技**。释放消耗爆发能量。

| 属性 | 说明 |
|---|---|
| 初始状态 | 无，需学习解锁 |
| 学习后等级 | 1 级 |
| 最高等级 | 6 级 |
| 升级所需等级 | 12/24/36/48/60 |
| 吟唱 | ❌ 不需要，直接生效 |

## 被动战技

简称**被动**。条件触发，持续生效。

| 属性 | 说明 |
|---|---|
| 初始状态 | 可能需要学习解锁 |
| 学习后等级 | 1 级 |
| 升级 | 通常不可升级 |
| 触发方式 | 随时判断条件是否成立 |

## 战斗天赋

根据角色定位获得的被动战技：

| 定位 | 可获得对应天赋 |
|---|---|
| 核心 / 先锋 / 近卫 / 支援 / 治疗 | 各自定位的战斗天赋 |

- 多定位角色可学习多个战斗天赋
- 但只能选择其中 **1 个** 生效
