# 决策点

决策点表示角色在回合内允许的**操作数量**。

> 决策点由游戏平衡常数设置，与角色能力值无关。每个角色的决策点是独立的。

## 决策点配额

默认情况下，回合内各类行动各限一次，称为"配额"：

| 行动 | 默认配额 |
|---|---|
| 普通攻击 | 1 |
| 战技 | 1 |
| 爆发技 | 1 |
| 魔法 | 1 |
| 物品 | 1 |

被动技能和特效可以操作决策点及其配额，赋予动态策略性。

::: info 临时全能配额
允许在没有对应配额时，将全能配额转换为指定配额使用。临时配额仅当回合有效。
:::

## 初始化与补充

| 规则 | 数值 |
|---|---|
| 初始决策点上限 | 2 |
| 回合开始时补充 | 上限的一半（向下取整，至少 1） |
| 每轮到自己 Turn | 上限 +1（最多 7） |
| 完全行动不能/行动受限 | 仅补充上限的 1/4（向下取整，至少 1） |

## 使用消耗

| 行动 | 消耗 |
|---|---|
| 普通攻击 | 1 决策点 |
| 战技 / 魔法 | 2 决策点 |
| 物品 | 1 决策点 |
| 回合内爆发技 | 2 决策点 |
| 回合外爆发技 | 3 决策点 |

## 硬直时间（多次行动）

当回合内行动次数 > 1 时：

```
回合结束时硬直时间 = 最长一次行动的硬直时间 + 本回合已使用的决策点数量
```

单次行动或放弃行动不适用此规则。

## 特殊情况

- 回合内吟唱魔法 → 直接结束回合
- 预释放爆发技 → 不适用上述规则
- 先行动再吟唱 → 硬直时间累计到吟唱结算的回合
