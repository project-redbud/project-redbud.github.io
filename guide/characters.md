# 角色概述

## 等级

角色最高等级为 **60 级**，每升 1 级所需经验逐级提高。

## 角色定位

游戏中角色按功能分为五个定位：

| 定位 | 英文 | 特点 |
|---|---|---|
| 核心 | Core | 极高输出，相对脆弱 |
| 先锋 | Vanguard | 中上等攻防，先发制人 |
| 近卫 | Guardian | 极高生命和护甲，专注保护 |
| 支援 | Support | 提供增益/减益，身板较脆 |
| 治疗 | Medic | 治疗和防御，身板较脆 |

## 能力值总览

角色能力值分为 **基础能力值** 和 **额外能力值**：

- **基础能力值**：基于初始属性、核心属性、等级计算
- **额外能力值**：通过装备、技能等方式获得

> 所有百分比加成的能力值乘算，都基于角色的基础能力值计算。

| 能力值 | 说明 | 有初始值 |
|---|---|---|
| 生命值 (HP) | 角色生存能力 | ✅ |
| 魔法值 (MP) | 释放魔法消耗 | ✅ |
| 爆发能量 | 战技/爆发技消耗，最大 200 | — |
| 行动速度 | 转化为行动系数 | ✅ |
| 攻击力 | 物理伤害基础 | ✅ |
| 物理护甲 | 转化为物理伤害减免 | ✅ |
| 魔法抗性 | 直接参与魔法伤害减免 | — |
| 加速系数 | 削减吟唱时间 | — |
| 闪避率 | 普通攻击闪避检定 | — |
| 暴击率 | 初始 5% | — |
| 暴击伤害 | 初始 125% | — |
| 物理穿透 | 削减目标护甲 | — |
| 魔法穿透 | 削减目标魔法抗性 | — |
| 冷却缩减 | 减少技能冷却时间 | — |
| 生命回复 | 时间流逝时回复 | ✅ |
| 魔法回复 | 时间流逝时回复 | ✅ |
| 攻击距离 | 战棋地图上的施法距离 | — |
| 移动距离 | 战棋地图上的移动范围 | — |
