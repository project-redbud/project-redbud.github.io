# 能力值详解

## 最大生命值

```
基础最大生命值 = 初始生命值 + (等级-1) × (17 + 0.68 × 初始生命值) + 基础力量 × 9
最大生命值 = 基础最大生命值 + (额外力量 × 9) + 额外生命值加成
```

## 最大魔法值

```
基础最大魔法值 = 初始魔法值 + (等级-1) × (1.5 + 0.14 × 初始魔法值) + 基础智力 × 8
最大魔法值 = 基础最大魔法值 + (额外智力 × 8) + 额外魔法值加成
```

## 爆发能量

- 最大值 200 点
- 释放战技和爆发技消耗
- 造成/受到伤害获得少量爆发能量

## 行动速度

```
行动系数 = 行动速度 / 1500 × 100%（保留两位小数）
```

行动系数上限 **90%**（速度 1350 时达到上限），用于减少硬直时间。

## 攻击力

```
基础攻击力 = 初始攻击力 + (等级-1) × (0.95 + 0.045 × 初始攻击力) + 基础核心属性
总攻击力 = 基础攻击力 + 额外攻击力加成
```

## 物理护甲

```
物理伤害减免 = 护甲 / (护甲 + 240) × 100%
```

## 加速系数

用于削减魔法吟唱时间。当角色吟唱魔法时，吟唱时间替代硬直时间插入顺序表，此时用加速系数（而非行动系数）来削减时间。

## 闪避率

仅对普通攻击有效。投掷 100 以内随机数，小于闪避率则闪避成功。

## 暴击率 & 暴击伤害

- 初始暴击率：**5%**
- 初始暴击伤害：**125%**
- 任何伤害计算减伤后，进行暴击检定

## 冷却缩减

减少所有主动技能（战技、魔法、爆发技）的冷却时间。

## 生命回复 & 魔法回复

时间流逝时按比例回复：

```
回复量 = 回复值 × 流逝的单位时间
```

例如生命回复 0.5，时间流逝 1 秒 → 回复 0.5。

## 生命偷取（吸血）

造成伤害时，按比例吸取生命值转化为治疗。

## 护盾值

- 区分物理护盾和魔法护盾
- 护盾生效期间替代生命值结算伤害
- 护盾耗尽后溢出伤害由生命值承担

## 攻击距离 & 移动距离

在战棋地图上使用曼哈顿距离。基础攻击距离由武器提供，基础移动距离由角色定位提供。
