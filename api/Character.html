Character ​

角色实体，位于 Milimoe.FunGame.Core.Entity。

使用 Factory.Get() 创建，并需要设置 InitRequired 标记的属性。

属性 ​

属性类型说明
GuidGuid唯一标识符
Namestring角色姓
FirstNamestring角色名
NickNamestring昵称
Levelint等级
HP / MaxHPdouble当前/最大生命值
MP / MaxMPdouble当前/最大魔法值
EPdouble当前爆发能量
STRdouble力量
AGIdouble敏捷
INTdouble智力
ATKdouble攻击力
DEFdouble物理护甲
SPDdouble行动速度
MagicTypeMagicType魔法属性
FirstRoleTypeRoleType角色定位
SkillsList<Skill>技能列表
EffectsList<Effect>当前状态栏特效
EquipSlotEquipSlot装备栏
UserUser所属玩家
TeamTeam所属队伍
IsAlivebool是否存活
IsDeadbool是否死亡
